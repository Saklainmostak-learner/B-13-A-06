# 🚀 DigiTools

## 📌 Description
DigiTools is a modern digital tools marketplace where users can explore, select, and purchase premium tools like AI writing, design templates, and productivity software. It provides a smooth shopping experience with cart functionality and a clean UI.

---

## ⚙️ Technologies Used
- React.js
- Tailwind CSS
- Vite
- JavaScript (ES6)
- React Toastify
- Lucide React Icons

---

## ✨ Features

### 🛒 1. Dynamic Cart System
Users can add/remove products from the cart and see the total price updated instantly.

### 🎯 2. Interactive UI
Modern responsive design with smooth transitions, clean layout, and mobile-friendly navigation.

### 🔔 3. Smart Notifications
Real-time toast notifications when adding/removing items or completing checkout.
